# case多条件分支语句

```shell
#介绍:
	在生产环境下要根据可能出现的状况来加载不同的结果,case则是来处理这种情况的

#语法:
	case $变量 in---若变量满足条件1则执行条件1代码...以此类推
	条件1)
		command1
	;;---每个代码块执行完毕要以;;作为结尾,换行开启下一个变量预案参数
	条件2)
		command2
	::
	...
	esac---此处代表结束,则是case倒过来拼写

#使用场景举例---判断内存使用率,命令参数等

#使用案例:
	read -p 'Number: ' num

	case $num in
	1 | One)---此处需要注意,用|可以分隔参数变量
		echo 'hello'
	;;
	2 | Two)
		echo 'hey'
	;;
	* | No)---*是shell的通配符,所以按照逻辑可以作为以上参数以外的特殊变量,也是shell编程中需要注意的一个功能
		echo 'Okay, bye'
	::
	esac
```
