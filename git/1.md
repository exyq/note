## Git本地操作

#### git基本操作
- init  **git初始化**
- status  **查看暂存区状态**
- add *File or Dir*  **添加git文件追踪**
- diff  **查看git文件修改前后的区别**
- reset  **让文件回到未被追踪状态**
- commit -m "commit message"  **提交暂存区**
- rm --cached *File or Dir*  **让git停止追踪已经追踪过的文件或文件夹**

#### git config
- --global user.name "*name*"  **添加全局用户名配置**
- --global user.email "*email* "  **添加全局用户邮箱配置**
- --global core.editor vim  **更改默认编辑器为vim**

#### .gitignore文件
- **让git停止追踪的文件列表**
