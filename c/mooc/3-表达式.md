## C语言表达式
- 表达式中含有**运算符**与**算子**两个元素
	- **运算符**(*operator*)是进行运算的动作,比如加法运算符与减法运算符
	- **算子**(*operaand*)指参与运算的值,这个值可以是常数,也可以是变量,还可以是一个方法的返回值
	- 例如 **a=b+c** 中的**a**,**b**,**c**是算子,**=**,**+**是运算符
- 运算符优先级
	| 优先级 | 运算符 | 运算 | 结合关系 | 举例 |
	| :---: | :---: | :---: | :---: | :---: |
	| 1 | + | 单目不变 | 自右向左 | a*+b |
	| 1 | - | 单目取负 | 向右自左 | a*-b |
	| 2 | * | 乘 | 向左自右 | a*b |
	| 2 | / | 除 | 自左向右 | a/b |
	| 2 | % | 取余 | 自左向右 | a%b |
	| 3 | + | 加 | 自左向右 | a+b |
	| 3 | - | 减 | 自左向右 | a-b |
	| 4 | = | 赋值 | 自右向左 | a=b |
	- 赋值也是运算,也有结果
	- **a=6**的结果是被赋予的值,也就是**6**
	- **a=b=6** --> **a=(b=6)**
- 复合运算符
	- **+ - * / %**,可以和 **=** 结合,形成复合运算符: **+= -= * = /= %=**
	- 复合运算符中间没有空格
	- 复合运算符是将赋值右方的值计算完后再将得出的值与赋值结果结合
		- 如total += (sum+100/2);
			- total = total+(sum+100)/2;
		- total *= sum+12;
			- total = total*(sum+12);
- 递增递减运算符
	- **++** **--** 是两个很特殊的运算符,并为单目运算,这个算子必须是变量. 这两个运算符分别叫递增和递减运算符,他们的作用就是给这个变量+1或-1
	- 如 **count++**;
		- count += 1;
			- count = count + 1;
	- **++** **--**可以放在变量后面和前面,放在前面叫做前缀形式,放在后面叫做后缀形式
	- **a++** 的值是**a+1**以前的值,而 **++a**是加了**1**以后的值,无论哪个, **a**自己的值都 **+1**了
	- 上面的话表达方式也可以理解为先后关系,a++是a, ++a是a+1
	- 设a=10,a++输出的是10,++a输出的是11
