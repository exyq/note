## C语言判断
```c
if (条件成立) {
	...
}
```
- 上述代码是if判断语句的基本语法
	- 表达的是如果条件成立则执行块中的语句
		- 如果条件不成立则跳过执行下面的语句
- 判断条件包含计算两个值之间的关系,叫做关系运算
### 关系运算符
| 运算符 | 意义 |
| :---: | :---: |
| == | 相等 |
| != | 不相等 |
| > | 大于 |
| >= | 大于或等于 |
| < | 小于 |
| <= | 小于或等于 |
- 关系运算的结果
	- 当两个值的关系符合关系运算符的预期时,关系运算的结果为整数1,否则为整数0
##### 关系运算的优先级
- 关系运算的优先级比算数优先级低,但比赋值运算的优先级要高
- 判断是否相等的 **==** 和 **!=** 的优先级比其他低,而连续的关系运算是从左到右进行的
##### 注释
- 注释(comment)是插入在程序代码中,用来向读者提供解释信息;他们在对于程序的功能没有任何影响,但是往往能使得程序更容易被人类读懂
```c
//单行注释(即从//开始到结尾都被计算机不予理会)
/*
	多行注释(即从开始到结束这段代码,计算机都不予理会)
*/
/*也可以用于一行内的注释*/
```
