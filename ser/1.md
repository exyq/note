# CentOS基础配置笔记

## DVD自动挂载并设定本地yum源
``` shell {.line-numbers}
blkid //查看DVD属性
mkdir /etc/yum.repos.d/bak /mnt/cdrom
mv /etc/yum.repos.d/* /etc/yum.repos.d/bak/
mv /etc/yum.repos.d/bak/CentOS-Media.repo /etc/yum.repos.d/
mount /dev/sr0 /mnt/cdrom
```
## 编辑源配置文件
``` vim
~vim /etc/yum.repos.d/CentOS-Media.repo
baseurl=file:///mnt/cdrom
#	file:///media/cdrom/
#	file:///media/cdrecorder/
...
enabled=1
```

## 配置自动挂载
``` shell {.line-numbers}
echo /dev/sr0 /mnt/cdrom iso9660 defaults 0 0 >> /etc/fstab
```

## 更新yum源
``` shell {.line-numbers}
sudo yum clean all
sudo yum makecache
```

## CentOS IP与网卡设定
``` vim
~ vim /etc/sysconfig/ifcfg-ens33
BOOTPROTO=static
IPADDR=<IPAddress>
DNS=<DNSAddress>
NETMASK=<NetMask>
GATEWAY=<GateWay>
ONBOOT=yes
```
### 重启网卡
``` shell {.line-numbers}
sudo systemctl restart network
```

## 编辑hosts文件
``` vim
~ vim /etc/hosts
<IPAddress> <HostName> <Domain> <OtherHostName> <OtherDomain>
```
### 查看hosts
``` shell {.line-numbers}
hostname -f
```
